Uji coba 2

HTML CSS JavaScript

Kesimpulan: Radio button dengan atribut name yang sama (misal "fav_language") akan membentuk grup sehingga hanya satu yang dapat dipilih. Jika name tidak diberikan, masing-masing radio dapat dipilih secara bersamaan karena tidak terkait. Jika name berbeda, masing-masing independen dan dapat dipilih semua. Percobaan ini membuktikan bahwa name berfungsi untuk mengelompokkan radio button.

Uji coba 3

Kesimpulan: Checkbox memungkinkan pemilihan lebih dari satu, terlepas dari atribut name. Name yang sama atau berbeda tidak mempengaruhi kemampuan memilih, namun berguna untuk pengiriman data ke server. Pada percobaan, semua checkbox dapat dicentang bersamaan.

Kesimpulan: Tombol "Click Me!" jika ditekan akan memunculkan alert "Hello World!". Input color menampilkan color picker, input date menampilkan date picker. Semua berfungsi sesuai deskripsi.

Uji coba 5





Kesimpulan: datetime-local memungkinkan memilih tanggal dan waktu lokal, month memilih bulan dan tahun, email melakukan validasi format email sederhana. Tampilan antarmuka bergantung pada browser yang digunakan.

Uji coba 6



Kesimpulan: Input file digunakan untuk memilih file dari komputer. Input hidden tidak ditampilkan namun nilainya akan dikirim bersama form, berguna untuk menyimpan data tersembunyi seperti ID.

Uji coba 7



Kesimpulan: Input number dengan min=1 max=10 membatasi nilai yang bisa dimasukkan. Jika atribut min diubah menjadi -50, maka input dapat menerima nilai negatif hingga -50. Range memberikan slider dengan rentang 0-50. Perubahan min pada number mempengaruhi batas bawah.

Uji coba 8





Kesimpulan: type="url" memvalidasi input harus berupa URL yang valid. type="search" mirip dengan text namun dengan gaya khusus di beberapa browser. type="tel" tidak melakukan validasi otomatis, namun atribut pattern dapat digunakan untuk validasi pola nomor telepon. Browser mungkin menampilkan pesan error jika pola tidak cocok saat submit.

Uji coba 9



Kesimpulan: type="time" menampilkan pemilih waktu, type="week" menampilkan pemilih minggu. Keduanya menyediakan antarmuka yang memudahkan pengguna memilih waktu dan minggu. Tampilan bervariasi antar browser.

Catatan: Untuk uji coba 2 dan 3, telah dilakukan percobaan dengan memberi atribut name sama dan berbeda sesuai skenario. Hasilnya dijelaskan pada kesimpulan masing-masing.
